Coffee Shop App

Description:
A modern coffee shop web app built using React and Vite. Users can browse the menu, see detailed product information, view testimonials, and download the app on Android or iOS.

Features

Landing Page

Hero section with animated coffee image

About section with coffee shop details

Animated sections using AOS (Animate On Scroll)

Menu / Services Page

Displays all available drinks and food items

Each product shows price and image

Click on Add to Cart to add items to the order

Product image click shows extra information card

Sequential numbering of products

Order Page

Shows all products added to the cart

Option to clear all items with a Clear button

App Download Section

Buttons for Google Play Store and Apple App Store

Testimonial Section

Customer reviews displayed in a carousel using react-slick

Fully responsive and animated

Footer

Company information and links

Social media icons (Facebook, Instagram, LinkedIn)

Technologies Used

React – Component-based frontend development

Vite – Fast build tool for React apps

HTML & CSS – Structure and styling

Tailwind CSS – Utility-first CSS framework for responsive design

JavaScript – Logic and interactivity

AOS – Scroll animations

Slick-carousel (react-slick) – Testimonials slider

Additional Notes

Products have cool font for names (you can customize in Tailwind)

Cart notification shows the number of items in the order

Smooth scrolling is implemented with CSS scroll-behavior: smooth

Animations are fully responsive and mobile-friendly
